Démarrage rapide

5 étapes pour être opérationnel en 15 minutes
1

Mon Entreprise

Paramètres → Mon Entreprise → renseigner SIRET, adresse, email. Ces données apparaissent sur tous vos PDFs.

2

Sélection Métier(s)

Paramètres → Sélection Métier(s) → cocher votre activité. La sidebar s'adapte automatiquement avec les modules de votre métier.

3

Catalogue Articles

Paramètres → Catalogue Articles → créer vos prestations (référence, libellé, prix HT, TVA). Au minimum 3-5 articles pour commencer.

4

Premier client + Animal

Clients → Nouveau → renseigner le client. Animaux → Nouveau → associer l'animal au client. Vous pouvez maintenant créer des séances et des devis.

5

Premier devis

Devis/Contrats → Nouveau → sélectionner le client, ajouter des lignes, enregistrer. Envoyez-le par email directement depuis PetSuite.

PDFs & Documents — Règle obligatoire

Toutes les éditions PDF s'ouvrent en modal
🚨 Règle absolue : tous les PDFs s'ouvrent dans une fenêtre modale (popup dans la page). Jamais d'ouverture dans un nouvel onglet, jamais de navigation vers une page PDF.

Composant DocumentActions (🖨️ 📧 🖊️)

🖨️ Bouton PDF

Ouvre le PDF dans une modal avec iframe. Boutons intégrés : "Ouvrir dans un onglet" et "Imprimer".

📧 Bouton Email

Modal avec sélecteur de destinataire (client + contacts du dossier), objet et message pré-remplis.

🖊️ Bouton Signature

Envoie le document sur Yousign. Le signataire reçoit un email avec lien de signature sécurisé.

Comptabilité & Export FEC

Conforme DGFiP — NF 525
1

Paramétrer les comptes comptables

Paramètres → Plan comptable → vérifier les comptes 411000 (clients), 706000 (ventes), 445710 (TVA), 512000 (banque).

2

Créer un exercice comptable

Comptabilité → Exercices → Nouveau → définir les dates de début et fin. PetSuite génère le FEC sur la période choisie.

3

Générer l'export FEC

Export FEC → Générer → choisir l'exercice. PetSuite génère le fichier avec hash SHA-256 par ligne (chaînage optionnel).

4

Vérifier l'équilibre

Vue v_fec_export_non_equilibre → doit être vide. Si des lignes apparaissent, une écriture est déséquilibrée.

✅ L'export FEC de PetSuite est conforme au format DGFiP (19 colonnes pipe-séparé). Chaîne de hash SHA-256 disponible pour les contrôles anti-fraude.
