Étape 1 Activer et démarrer Eliot
Eliot est une option disponible dans les formules Pro et Expert.
  • Paramètres → Modules → Eliot IA → Activer
  • Le bouton Eliot (icône robot 🤖) apparaît en bas à droite de toutes les pages
  • Cliquer sur le bouton → fenêtre de chat s'ouvre
  • Taper votre première demande en langage naturel
  • Eliot répond en moins de 3 secondes avec l'action effectuée ou une demande de précision
  • Fermer la fenêtre → Eliot se referme, vous restez sur la même page
💡 Eliot est une option commerciale. Le logiciel fonctionne parfaitement sans lui.

Étape 3 Eliot — Création de séances et devis
Créez des documents complets en une phrase.

  • 'Génère un devis pour Marie pour 5 séances de suivi à 70€ chacune'
  • → Devis DEV-2026-0031 créé : 5 lignes × 70€ = 350€ HT · 420€ TTC
  • 'Duplique le dernier devis de Marie avec une remise de 10%'
  • → Devis dupliqué avec remise appliquée sur toutes les lignes

Étape 6 Ce qu'Eliot ne fait jamais
Les limites strictes d'Eliot pour protéger vos données.
  • ❌ Supprimer des données (clients, animaux, factures, séances)
  • ❌ Modifier le code source du logiciel
  • ❌ Exécuter du SQL direct dans la base de données
  • ❌ Modifier le schéma de la base de données
  • ❌ Accéder aux données d'autres comptes
  • ❌ Contourner les règles NF525 (numérotation, inaltérabilité)
  • ✅ Toutes ses actions passent par les mêmes API sécurisées que l'interface normale
🔒 La sécurité d'Eliot est garantie par architecture : il ne peut pas faire ce que l'interface ne permet pas.
