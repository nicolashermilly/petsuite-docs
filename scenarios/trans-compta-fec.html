Étape 1 Configuration du plan comptable
Avant de commencer, vérifiez votre plan comptable avec votre expert-comptable.
  • Paramètres → Plan comptable → vérifier les comptes par défaut
  • 706100 — Prestations comportementalistes
  • 706200 — Prestations toilettage
  • 707000 — Ventes de marchandises (produits)
  • 411000 — Clients (créances)
  • 445710 — TVA collectée 20%
  • 512000 — Banque (BNP Paribas)
  • 401000 — Fournisseurs
  • Adapter si votre comptable utilise une sous-numérotation différente
💡 Demandez à votre expert-comptable le plan comptable exact qu'il souhaite utiliser. Le paramétrage initial est crucial.
2
Étape 2 Création de l'exercice comptable
Créez l'exercice avant toute première facture.
  • Comptabilité → Exercices → + Nouvel exercice 2026
  • Date début : 01/01/2026 · Date fin : 31/12/2026
  • Code exercice : EX2026 · Libellé : Exercice comptable 2026
  • Marquer comme exercice courant (toutes les nouvelles pièces s'y rattachent)
  • Vérifier que l'exercice précédent (2025) est bien clôturé si applicable
3
Étape 3 Écriture automatique — Vente d'une prestation
À chaque facture validée, Anim'Gest génère l'écriture comptable.
  • Facture FAC-2026-0001 — Dupont Marie — 756,00€ TTC émise
  • Écriture générée automatiquement :
  • Débit 411001 Dupont Marie 756,00€
  • Crédit 706100 Prestations comportementalistes 630,00€
  • Crédit 445710 TVA collectée 20% 126,00€
  • Équilibre : 756 = 630 + 126 ✅
  • Référence pièce : FAC-2026-0001 · Journal : VTE (ventes)
4
Étape 4 Écriture automatique — Règlement client
Quand le client paye, l'écriture de trésorerie est générée.
  • Règlement CB 756,00€ enregistré le 15/05/2026
  • Écriture générée :
  • Débit 512000 Banque 756,00€
  • Crédit 411001 Dupont Marie 756,00€
  • Journal : BNQ (banque)
  • Le compte 411001 (Dupont Marie) est soldé → aucun impayé résiduel
5
Étape 5 Consultation de la balance
En cours d'exercice, vérifiez régulièrement votre balance.
  • Comptabilité → Balance → Exercice 2026 · Période : janvier-mai
  • Pour chaque compte : Débit cumulé, Crédit cumulé, Solde
  • Compte 411 : solde débiteur = impayés clients (cible : le moins possible)
  • Compte 512 : solde débiteur = trésorerie disponible
  • Compte 706 : solde créditeur = CA prestations (le croissant)
  • Vérification globale : Total Débit = Total Crédit ✅
  • Exporter la balance en PDF pour votre comptable
6
Étape 6 Génération du FEC
En fin d'exercice, générez le Fichier des Écritures Comptables.
  • Comptabilité → Export FEC → Exercice 2026 → Générer
  • Fichier TXT généré : FEC_2026_AnimGest.txt
  • 18 colonnes obligatoires : JournalCode, JournalLib, EcritureNum, EcritureDate, CompteNum...
  • Vérification automatique : Total Débit = Total Crédit ✅
  • Taille fichier : environ 450 lignes pour une activité standard
  • Télécharger le fichier et le transmettre à votre expert-comptable
⚖️ Le FEC est exigible par l'administration fiscale en cas de contrôle. Délai de conservation : 10 ans.
7
Étape 7 Vérification qualité du FEC
Avant de transmettre, vérifiez la qualité du fichier.
  • Ouvrir le FEC dans Excel : vérifier que les 18 colonnes sont présentes
  • Colonne Debit + colonne Credit : calculer les sommes (=SOMME)
  • Les deux totaux doivent être strictement égaux
  • Vérifier que la colonne EcritureNum est séquentielle (pas de trou)
  • Vérifier que les dates sont au format AAAAMMJJ (ex : 20260515)
  • Vérifier que la colonne CompteNum commence toujours par un chiffre
8
Étape 8 Transmission et clôture de l'exercice
Transmettez le FEC et clôturez l'exercice comptable.
  • Envoyer le FEC à votre expert-comptable par email sécurisé ou plateforme partagée
  • Joindre : balance des comptes PDF + relevés bancaires de l'année
  • Joindre : état des immobilisations si applicable
  • Attendre la validation de votre comptable avant de clôturer
  • Comptabilité → Exercices → EX2026 → Clôturer l'exercice
  • Une fois clôturé : aucune nouvelle écriture possible sur cet exercice
  • Créer immédiatement l'exercice 2027

ETAPE 9 — Export TVA par taux

Accédez à Facturation → Export TVA pour générer un état détaillé par taux sur la période de votre choix.

  • → Sélectionnez la période (raccourcis : mois courant, trimestre, année)
  • → Visualisez la répartition par taux : 0 % / 5,5 % / 10 % / 20 %
  • → Exportez en CSV (UTF-8 BOM, séparateur ;) compatible Excel
  • → Transmettez à votre expert-comptable ou intégrez dans votre déclaration CA3

Route API : GET /tva/export?date_debut=YYYY-MM-DD&date_fin=YYYY-MM-DD&format=csv

10

ETAPE 10 — Notes de frais

Gérez vos dépenses professionnelles depuis Comptabilité → Notes de frais.

  • → Créez une note de frais (numérotation auto NF-YYMM-NNNN)
  • → Saisissez les lignes de dépenses avec catégorie PCG (carburant, repas, matériel...)
  • → Le calcul HT/TVA est automatique
  • → Workflow statut : Brouillon → Soumise → Validée → Remboursée
  • → Export CSV pour intégration comptable
