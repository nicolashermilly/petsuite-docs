01 Étapes du scénario
01 Configurer le cabinet vétérinaire
  • Paramètres → Mon métier : sélectionner Vétérinaire
  • Numéro Ordre des Vétérinaires, SIRET, espèces traitées
  • Configurer les taux TVA : 0% médicaments remboursables, 5,5% vermifuges, 10% prestations, 20% cosmétiques
  • Paramétrer les types de consultation : examen, vaccination, chirurgie, urgence, téléconsultation
  • Convention de téléconsultation conforme au Code de déontologie vétérinaire
02 Créer le dossier client et patient
  • Clients → Nouveau client : propriétaire
  • Fiche animal : espèce, race, sexe, âge, poids, identification (puce / tatouage)
  • Antécédents médicaux, allergies, traitements en cours
  • Historique complet des consultations visible depuis la fiche animal
  • Importation des données ICAD via numéro de puce
03 Planifier les consultations
  • Agenda → Nouvelle séance : type Consultation ou Vaccination
  • Durées estimées : consultation (20 min), urgence (variable), chirurgie (1-2h)
  • Rappels automatiques J-1 par SMS/email
  • File d'attente spécifique pour les urgences sans RDV
  • Vue jour : tableau de bord du planning du cabinet
04 Conduire la consultation
  • Ouvrir la séance depuis l'agenda : noter le motif de consultation
  • Examen clinique : observations dans les notes de séance
  • Diagnostics, actes réalisés, prescriptions
  • Suivi des constantes (poids, température, FC, FR) avec graphique d'évolution
  • Photos cliniques dans la GED (lesion, radio, échographie)
05 Prescriptions et ordonnances
  • Générer l'ordonnance depuis la séance : médicaments, posologie, durée
  • Stock médicaments débité automatiquement si dispensé au cabinet
  • Ordonnances PDF datées, signées et stockées dans la GED du patient
  • Renouvellements : alertes automatiques dans l'agenda
  • Conformité réglementation AMM — registre médicaments tenu à jour
⚠ Attention : Ordonnances vétérinaires soumises à la réglementation AMM. Tenir à jour le registre des médicaments classés.
06 Gestion du stock médicaments
  • Paramètres → Articles : créer les médicaments avec taux TVA correct
  • Seuil d'alerte et réapprovisionnement automatique
  • Achats → Bon de commande pour le réapprovisionnement laboratoire
  • Inventaire périodique obligatoire des médicaments sous contrôle
  • Traçabilité lots et numéros de série pour produits sensibles
07 Facturation multi-TVA
  • Factures → Nouvelle facture depuis la consultation
  • Taux TVA par ligne : 0% (médicaments remboursés), 5,5% (vermifuges), 10% (consultation), 20% (cosmétiques)
  • Export TVA détaillé par taux — indispensable pour déclaration CA3
  • Avoir client si remboursement (assurance animaux)
  • Paiement échelonné possible via portail client (chirurgies coûteuses)
08 Urgences, statistiques et comptabilité
  • Séance « Urgence » en dehors des plages habituelles — majoration configurable
  • Compte-rendu d'urgence transmis automatiquement au vétérinaire traitant
  • Statistiques : actes les plus fréquents, CA par espèce, taux de vaccination
  • Suivi des impayés avec relances automatiques
  • Export TVA + FEC pour expert-comptable
  • Notes de frais : formations, congrès vétérinaires, matériel, cotisation ordinale
