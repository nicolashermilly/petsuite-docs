🔌 Architecture technique

⚠️ Filtrage côté frontend : la route GET /api/questionnaires retourne tous les questionnaires. Le filtrage par ?metier=X est actuellement appliqué côté client. Une route serveur filtrée est prévue en S45.

Route API

GET /api/questionnaires         → tous les questionnaires
GET /api/questionnaires?metier=COMPORTEMENTALISTE → filtrés
POST /api/questionnaires         → création
PUT  /api/questionnaires/:id    → mise à jour
DELETE /api/questionnaires/:id  → suppression
