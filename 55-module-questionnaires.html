📋 Module Questionnaires

Modèles de questionnaires par métier — 25 modèles pré-définis pour 11 métiers

Page principale

Route : /app/questionnaires — accessible depuis chaque métier dans la sidebar

Passage du métier via query string : /app/questionnaires?metier=PET_SITTER

Fonctionnalités

  • Grille 3 colonnes de cartes cliquables
  • Filtre par métier (boutons toggle)
  • Barre de recherche texte (titre + description)
  • Chaque carte : titre, description, nb questions, badge métier, boutons Aperçu/Envoyer
