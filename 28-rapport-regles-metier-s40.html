Conventions BDD

Format des commentaires — obligatoire sur toutes les colonnes

Format clé étrangère (FK)

COMMENT 'Relation NomTable — description'

-- Exemples :
COMMENT 'Relation tiers — client propriétaire de l''animal'
COMMENT 'Relation tva_taux — taux TVA applicable à la ligne'
COMMENT 'Relation seance — séance parente de cette ligne'

Format colonne standard

COMMENT 'Description courte du champ'

-- Exemples :
COMMENT 'Montant HT calculé par trigger (QTE × PU_HT)'
COMMENT 'Statut RFE : DEPOSEE_PDP|TRANSMISE|ACCEPTEE|EN_LITIGE|REFUSEE'
COMMENT 'Flag géniteur : 1=actif reproduction, 0=non géniteur'
⚠️ Priorité S41 : Audit des colonnes sans commentaire sur les tables principales (facture, facture_ligne, seance, animal, contact, tiers). Script Python de vérification à créer.
