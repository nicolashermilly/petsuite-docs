01 Profil entreprise

Vos coordonnées commerciales — alimentent vos PDF et signatures.

Informations de base

Renseignez votre identité commerciale — ces données alimentent automatiquement vos PDF, factures et signatures email.

  • Paramètres → Mon entreprise : raison sociale, SIRET, code APE, numéro TVA intracom
  • Adresse complète du siège (géolocalisée automatiquement)
  • Coordonnées de contact : téléphone, email principal, site web
  • Capital social, RCS, numéro RNE si applicable
  • Logo principal : format PNG transparent recommandé (512×512 px)
02 Métier et activités

Sélection du métier ou multi-métier.

Configurer votre activité principale

  • Paramètres → Mon métier : sélectionner parmi les 11 métiers disponibles
  • Multi-métier : si vous exercez plusieurs activités, activer toutes celles concernées
  • Chaque métier active automatiquement : ses spécificités d'agenda, ses types de séance, ses registres réglementaires, ses dashboards CEO
  • Pour comportementaliste/éleveur : ACACED, BPJEPS, numéro de capacité
  • Pour vétérinaire : numéro Ordre, certifications spécifiques
03 Charte graphique

Couleurs, logo, favicon, police.

Personnaliser l'interface aux couleurs de votre marque

  • Paramètres → Charte graphique", "Couleur primaire, secondaire, accent (palette complète)
  • Mode clair et mode sombre auto-générés à partir de votre palette
  • Logo header (apparaisant en haut de chaque page UI) : SVG ou PNG < 200 Ko
  • Favicon : 32×32 px PNG ou ICO
  • Police personnalisée (Google Fonts uniquement)
  • Prévisualisation en temps réel avant validation
  • 49 colonnes app_charte_graphique personnalisables
04 Éditions PDF

Personnalisation des PDF générés.

Personnaliser vos PDF (devis, factures, contrats)

  • Paramètres → Éditions", "Logo facturé (peut être différent du logo header)
  • Filigrane optionnel : PROFORMA, COPIE, ANNULÉ
  • CGV : téléverser votre PDF (sera ajouté en dernière page de chaque PDF)
  • Pied de page personnalisé : mentions légales, signature
  • Couleur primaire des PDF (bandeau en-tête)
  • Format précis : A4, marges 15mm, police standard
05 GED Documents

Chemins, sauvegardes, conservation légale.

Configurer la GED (Gestion Électronique de Documents)

  • Chemin du dossier GED : C:\AnimGest_GED\SCT_001\ (configurable)
  • Sauvegarde automatique quotidienne sur NAS RAID 6
  • Sauvegarde secondaire hebdomadaire (G:\ ou cloud chiffré)
  • Rotation : 30 jours/12 semaines/12 mois/7 ans
  • Conservation légale 10 ans pour factures et contrats
06 SMTP & SMS

Email et SMS transactionnels.

Configurer l'envoi d'emails (SMTP)

  • Paramètres → SMTP : configurer votre serveur d'envoi
  • Recommandé : Brevo (ex-Sendinblue) 300 emails/jour gratuit, Gmail SMTP avec mot de passe d'application
  • Adresse expéditeur (visible par vos clients)
  • Nom expéditeur (par exemple "Cabinet Dr Martin")
  • Bouton "Tester l'envoi" pour valider la configuration

SMS (optionnel)

  • Paramètres → SMS : Free Mobile API, OVH, Twilio
  • Utilisé pour les rappels de RDV J-1 et l'OTP signature Yousign
  • Coût typique : 0.05€ à 0.10€ par SMS selon opérateur
07 Stripe paiements

Encaissement CB en ligne.

Activer les paiements en ligne

  • Créer un compte Stripe sur stripe.com (validation 2-3 jours ouvrés)
  • Paramètres → Stripe : coller votre STRIPE_SECRET_KEY (mode live)
  • Webhook URL : https://votre-domaine/api/webhooks/stripe
  • Frais Stripe : 1.4% + 0.25€ par transaction CB européenne
  • Active : paiement direct facture, abonnements, paiement échelonné 3x/4x
08 Yousign signatures

Signatures électroniques qualifiées eIDAS.

Activer la signature électronique qualifiée

  • Créer un compte Yousign sur yousign.com (essai gratuit 14 jours)
  • Paramètres → Yousign : coller votre clé API
  • Choisir le niveau : Simple (SES), Avancé (AES), Qualifié (QES eIDAS)
  • Tarif typique : 0.50€ à 2€ par signature selon niveau
  • Active : signature devis, contrats, mandat SEPA, autorisations
09 Calendly webhook

Réservations en ligne automatiques.

Recevoir des réservations en ligne via Calendly

  • Paramètres → Calendly : copier l'URL webhook
  • Dans Calendly.com → Settings → Integrations → Webhooks → + Nouveau
  • URL : URL webhook copiée — Événement : invitee.created
  • Mapping automatique de 9 slugs Calendly → types de séance Anim'Gest
  • Voir le scénario Webhook Calendly pour les détails complets
10 Eliot IA

Assistant intelligent Anthropic.

Activer l'assistant IA Eliot

  • Créer un compte Anthropic sur console.anthropic.com
  • Paramètres → Eliot : coller la clé API sk-ant-...
  • Choisir le modèle : Claude Sonnet 4 (rapide, économique) ou Opus 4 (avancé)
  • Personnaliser : nom de l'assistant, avatar (parmi 20), voix synthèse vocale
  • Définir les limites : budget mensuel max, métier autorisé (selon votre licence)
  • Voir Référence avancée Eliot pour les 22 outils disponibles
11 Utilisateurs et rôles

Accès équipe avec permissions RBAC.

Créer les accès de votre équipe

  • Paramètres → Utilisateurs : nouvel utilisateur (email + rôle)
  • Rôles RBAC disponibles : ADMIN (accès total), GESTION (métier complet), CONSULT (lecture seule), COMPTA (comptabilité uniquement)
  • Personnaliser les permissions par module dans la matrice accès
  • L'utilisateur reçoit un email d'invitation pour définir son mot de passe
  • Activation/désactivation possible à tout moment
  • Journal d'audit : traçabilité des actions sensibles par utilisateur
