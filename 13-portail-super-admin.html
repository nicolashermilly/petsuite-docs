🏢 Vue d'ensemble

Le portail Super Admin permet à l'éditeur de PetSuite de gérer l'ensemble des instances déployées (multi-tenants). Chaque tenant correspond à une structure professionnelle cliente.

⚙️ Fonctionnalités

Gestion des tenants

Création, activation, suspension. Paramétrage des limites (utilisateurs, stockage GED, nombre d'animaux).

Monitoring

Tableau de bord global : tenants actifs, volumes de données, état des sauvegardes, alertes.

Conformité NF525

Vue consolidée de l'état de conformité NF525 par tenant. Alerte si anomalie détectée.

Déploiements

Suivi des versions déployées par tenant. Historique des mises à jour.

🔐 Accès

Le portail Super Admin est accessible uniquement depuis le réseau interne (192.168.1.x). Authentification renforcée avec JWT + 2FA. Rôle : SUPER_ADMIN uniquement.
