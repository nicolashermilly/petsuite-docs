02Authentification
Magic link — accès sans mot de passe
✉️
Principe du magic link
1
Le professionnel envoie l'invitation
Depuis la fiche client dans l'app : bouton "Inviter au portail". Email envoyé au propriétaire avec lien unique.
2
Clic sur le lien
Lien valable 7 jours. Redirige vers portail.{sous-domaine}.Anim'Gest.fr — authentification automatique.
3
Session créée
JWT client (rôle CLIENT) stocké en localStorage. Durée 30 jours. Renouvellement automatique si actif.
4
Reconnexion ultérieure
L'email du propriétaire suffit — nouveau magic link envoyé en 1 clic depuis la page de connexion.
🔐
Sécurité du magic link
TokenUUID v4 aléatoire — 128 bits d'entropie
StockageHash SHA-256 en BDD — pas le token brut
Validité7 jours maximum — usage unique (invalidé après clic)
ScopeLié au TIE_ID du propriétaire + tenant_id
HTTPSObligatoire — jamais en HTTP
Rate limit3 demandes/heure par email — anti-spam
Intégration Stripe existante
Stripe est déjà intégré dans le projet (Sprint 4). Le portail client réutilise ce module pour le paiement des factures — pas de nouveau développement requis.
💳
Flux de paiement
DéclencheurBouton "Payer" sur une facture ÉMISE
RedirectionStripe Checkout — page hébergée Stripe
MoyensCB, Apple Pay, Google Pay — selon config Stripe
RetourWebhook Stripe → INSERT reglement + statut FAC → PAYÉE
ConfirmationEmail automatique propriétaire + professionnel

URL du portail
Domaineportail.{sous-domaine}.Anim'Gest.fr — séparé de l'app métier
Exempleportail.cabinetmartin.Anim'Gest.fr
Couvert SSLOui — wildcard *.Anim'Gest.fr couvre portail.*
Routing TraefikHostRegexp({sub:[a-z0-9-]+}.Anim'Gest.fr) → détecte portail. vs app principale
Sécurité & isolation
🔒
Isolation des données
JWT clientContient : TIE_ID + tenant_id + rôle CLIENT
VérificationChaque requête vérifie que l'animal appartient au TIE_ID du JWT
Pas de croisementImpossible d'accéder aux animaux d'un autre propriétaire
Pas de données métierMontants HT, marges, commentaires internes masqués en requête SQL
Lecture seuleAucun endpoint de modification — méthode GET/POST paiement uniquement
🆕
