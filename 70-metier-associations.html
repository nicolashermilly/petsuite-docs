🤝 Associations

1

ETAPE 1 — Configurer le module Associations

  • → Paramètres → Mon métier : sélectionner ou activer Associations
  • Renseigner les informations de l’association : nom, SIRET, adresse, email de contact
  • Définir les tarifs d’adhésion : tarif normal, tarif réduit, tarif famille, durée de validité
  • Générer le token d’adhésion public depuis Associations → Paramétrage adhésion
Conseil : L’URL publique du formulaire d’adhésion peut être partagée sur votre site web ou réseaux sociaux avant même votre premier événement.
2

ETAPE 2 — Créer les premiers membres

  • → Associations → Membres → + Nouveau membre
  • Saisir : nom, prénom, email, téléphone, date de naissance (optionnel)
  • Le numéro de membre est attribué automatiquement en séquence
  • Alternative : partager le formulaire public — les inscriptions arrivent directement en attente de validation
Import : Pour un club existant, importez votre liste de membres via CSV depuis la page Membres.
3

ETAPE 3 — Enregistrer les adhésions

  • → Associations → Adhésions → + Nouvelle adhésion
  • Sélectionner le membre, choisir le tarif et saisir la date de début
  • La date de fin est calculée automatiquement selon la durée du tarif
  • Statut initial : En attente → passer à Actif après règlement
  • Renouvellement : bouton Renouveler dans la fiche adhésion
Suivi : Le tableau de bord Associations affiche en temps réel le nombre d’adhésions actives et celles expirant dans les 30 jours.
4

ETAPE 4 — Générer les reçus fiscaux

  • → Associations → Reçus fiscaux → + Nouveau reçu
  • Sélectionner le membre, saisir le montant et la nature : cotisation ou don
  • Le reçu est généré au format Cerfa 11580*03 conformément à la réglementation
  • Envoi par email directement depuis la fiche reçu
  • Numérotation automatique et archivage dans la GED
Fiscalité : Le reçu Cerfa permet au membre de déduire jusqu’à 66% du montant versé de son impôt sur le revenu (dans la limite de 20% du revenu imposable).
5

ETAPE 5 — Organiser un événement

  • → Associations → Evénements → + Nouvel événement
  • Renseigner : titre, date, lieu, description, capacité maximale, tarif d’inscription
  • Statut initial : Ouvert — les inscriptions sont acceptées
  • Inscrire les membres manuellement ou activer le formulaire public d’inscription
  • La jauge de capacité se met à jour à chaque nouvelle inscription
Types d’événements : Stage, conférence, portes ouvertes, concours, sortie terrain, assemblée générale — tout type d’événement associatif est gérable.
6

ETAPE 6 — Gérer les inscriptions et le check-in

  • Dans la fiche événement, liste des inscrits avec statut : Inscrit / Présent / Absent
  • Bouton Check-in sur chaque participant pour valider la présence le jour J
  • Compteur de présents mis à jour en temps réel
  • Export de la liste des participants pour la feuille d’émargement
Sur mobile : Le check-in fonctionne parfaitement depuis un smartphone — idéal pour valider les présences à l’entrée de l’événement.
7

ETAPE 7 — Coordonner les bénévoles avec le kanban

  • → Associations → Tâches : kanban visuel par statut
  • Colonnes : À faire — En cours — Terminé
  • Créer une tâche : titre, description, responsable, échéance, priorité
  • Glisser-déposer les cartes pour faire avancer les tâches
  • Exemples : préparation des salles, communication réseaux, logistique matériel
Organisation : Créez des tâches spécifiques à chaque événement pour garder une vue globale de l’avancement des préparatifs.
8

ETAPE 8 — Utiliser le formulaire d’adhésion public

  • → Associations → Paramétrage adhésion : copier l’URL publique
  • Partager sur : site web, page Facebook, Instagram, email, QR code imprimé
  • Le candidat remplit le formulaire : nom, prénom, email, téléphone, tarif souhaité
  • La demande apparaît dans Membres en attente de validation
  • Valider → le membre est créé et son adhésion enregistrée automatiquement
Sans saisie : Le formulaire public élimine toute ressaisie manuelle — le membre saisit lui-même ses informations.
9

ETAPE 9 — Suivre les indicateurs clés

  • → Associations (page d’accueil du module) : tableau de bord KPIs
  • Membres totaux, adhésions actives, événements ouverts, tâches en cours
  • Alertes adhésions expirant dans les 30 jours
  • Historique des renouvellements et taux de fidélisation
Reporting : Exportez les listes d’adhérents et les statistiques en CSV pour vos rapports d’assemblée générale.
10

ETAPE 10 — Comptabilité et clôture annuelle

  • Cotisations et dons : intégrés dans le bilan via les reçus fiscaux générés
  • Notes de frais bénévoles : saisir dans Comptabilité → Notes de frais avec justificatif
  • Export CSV des adhésions et reçus pour le commissaire aux comptes
  • Générer les reçus fiscaux annuels pour tous les membres en un clic
  • Archive GED : tous les documents sont conservés et horodatés
Conformité : Les reçus Cerfa 11580*03 sont générés conformément aux obligations fiscales des associations reconnues d’intérêt général.
