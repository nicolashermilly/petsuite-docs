⚠️ Litige livraison

  • Quantités reçues ≠ commandées
  • Produits endommagés
  • Demande d'avoir partiel ou total
  • Retour fournisseur

📡 Facture via PDP

  • Facture reçue par votre PDP (Pennylane)
  • Import automatique dans Anim'Gest
  • Contrôle vs bon de commande
  • Validation ou mise en litige

💳 Règlement SEPA

  • Virement SEPA fournisseur
  • Génération fichier SEPA XML
  • Lettrage avec la facture
  • Export comptable FEC
