⚙️ Règles critiques de déploiement

1
cd deploy AVANT tout docker composeToujours se placer dans le dossier deploy/ avant d'exécuter une commande Docker Compose.
2
.\COPIER_TOUTES_PAGES_v2.ps1 AVANT chaque build frontendCopie tous les fichiers page.tsx dans le contexte Docker. Sans ça, les nouvelles pages ne sont pas incluses dans l'image.
3
Séquence rebuild frontenddocker compose build frontend --no-cache && docker compose up -d frontend && docker compose restart nginx
4
Séquence rebuild API (après modif custom-routes.ts)docker compose build api --no-cache && docker compose up -d api && docker compose restart nginx
5
inject_v4_final.js = NE JAMAIS UTILISERContient un orphaned catch block → SyntaxError → crash loop API.
6
Nginx strip /api/ → routes sans préfixe dans custom-routes.tsNginx passe les requêtes /api/xxx à l'API comme /xxx. Ne jamais préfixer les routes avec /api/ dans le code.
7
DbService inaccessible après app.listen() → mysql2/promise directToutes les routes custom utilisent mysql2/promise avec les env vars DB_HOST, DB_PORT, DB_USER, DB_PASSWORD, DB_NAME.
8
PDF/Editions = TOUJOURS en Modal/PopupFactures : fetch('/api/editions/facture/${id}') → blob → <iframe> dans modal. Jamais de navigation directe.
9
Ne jamais écraser layout.tsxToujours patcher uniquement les 2 lignes nécessaires (import + usage). En cas de doute : git restore src/app/app/layout.tsx.
10
Colonnes sans DEFAULT → toujours NOW() explicite dans les INSERTÉvite les erreurs Field doesn't have a default value.

⚛️ Règles frontend (Next.js 15)

F1
"use client" — TOUJOURS première ligneToutes les pages App Router avec hooks ou événements doivent commencer par "use client";.
F2
import {{ api }} from "@/lib/api" — named export, jamais default
F3
Pas d'apostrophes françaises dans les strings TypeScriptUtiliser &apos; ou l'apostrophe droite ' dans le JSX.
F4
Leaflet en Next.js → charger dans useEffect avec guard mapInstanceRef.currentÉvite les doubles initialisations de carte en mode StrictMode.
F5
Leaflet CDN → cdnjs.cloudflare.com (pas unpkg)https://cdnjs.cloudflare.com/ajax/libs/leaflet/1.9.4/leaflet.min.js
